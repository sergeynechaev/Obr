«Обрамлятор» - программа для оформления ссылок на фотографии тегами IMG
с заданными настройками.

--------------------------
Программа написана на JavaScript-е, инсталляции не требуется.
Для локального использования просто распакуйте архив в любую папку
и откройте через интернет-браузер файл index.html
--------------------------

Версия 2.2.2 beta
Протестирована в браузерах Chrome, Firefox, IE, Opera, Safari.

Актуальную версию вы всегда сможете найти по адресу
http://dert.ru/obr/

Разрешается свободное использование и распространение при условии
сохранения авторства, копирайтов и ссылок на сайты автора.
